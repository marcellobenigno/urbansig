# UrbanSIG - Sistema de Geoinformação Voltado para a Gestão de Prefeituras


O **UrbanSIG** é um projeto de aprendizagem de desenvolvimento de aplicações WebGIS, utilizando as seguintes tecnologias:

- [Python][1]
- [GeoDjango][2]
- [PostGIS][3]
- [GeoServer][4]
- [Leaflet][5]
